Process

How it works?

Organize your kitchen inventory with the handly app on your phone using a built-in barcode scanner! Get recommended recipes based on your kitchen inventory. Get alerts when you run out of your favorite foods.

Sign Up

Sign up with your email address and a password. Build an inventory that belongs to you! Get alerts and recipes on the dashboard page.

Create Inventory

Click on the add items button to start building your inventory. Use the barcode scanning function to help adding items or manually add items using the input form.

Update

Visualize your items on the pantry page. Click on each item to update or delete. Get alerts and recipes on the dashboard page.

Get Recipes

Get fun, easy recipes using selected items or the entire inventory. Recipe will pop up on broswer for more details.

Features

Make an awesome meal with what you love!

KitchenFox is a sleek application that allow users to build, update and visualize their kitchen inventory on-the-go. These following features were built with optimizing the users' experience...

Easy Navigation

Utilized Stack Navigation to organize routes. User can roam around easily with the footer navagation bar. Designed to be intuitive and easy to use.

Barcode Scanning Functionality

The barcode scanner is built to recognize UPC barcodes connecting to a substantial UPC databases.

Recipes Recommedation

Recipes are recommended based on inventory and or selected items provided by the users.
